# Название проекта

Советчик по выбору карьеры и профессии

## Инструменты для реализации

- Язык программирования: Python
- Фреймворки/библиотеки:telebot (для работы с Telegram Bot API), pandas (для загрузки и обработки данных из CSV‑файла)
- БД:CSV‑файл (professions.csv) в качестве простой базы данных с информацией о профессиях
- Прочее:не используется

## Проект состоит из

1. Модуль 1:Модуль старта и навигации (команды /start и главное меню): приветствует пользователя, отображает главное меню с кнопками, обеспечивает базовую навигацию по функционалу бота.
2. Модуль 2:Модуль подбора профессий (сценарий «Найти профессию»): проводит опрос пользователя (интересы → сильные стороны), подбирает и показывает 3 подходящие профессии на основе данных из базы с указанием зарплаты, образования и навыков.
3. Модуль 3:Модуль справочной информации (сценарии «Узнать о профессиях подробнее» и «Помощь»): выводит полный список доступных профессий с кратким описанием каждой либо отображает справку по использованию бота.

## Ключевые цели

- Цель 1.Предоставить персонализированные рекомендации по профессиям на основе ответов пользователя на простые вопросы об интересах и сильных сторонах.
- Цель 2.Упростить поиск подходящей профессии для пользователей, которые не уверены в выборе карьерного пути, за счёт интерактивного диалога в мессенджере.
- Цель 3.Информировать о профессиях — дать пользователю быстрый доступ к структурированным данным (описание, зарплата, образование, навыки) как в виде рекомендаций, так и в виде общего списка.
