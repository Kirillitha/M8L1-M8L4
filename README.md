#ProfessionalBot — Telegram‑бот для подбора профессий

Интерактивный бот для Telegram, помогающий пользователям подобрать профессию на основе интересов и навыков. Бот предоставляет детальную информацию о профессиях, позволяет сохранять избранные варианты, просматривать статистику и оставлять обратную связь. Идеален для профориентации школьников, студентов и всех, кто задумывается о смене карьеры.

## Функционал

Бот предлагает следующие возможности:

* **Подбор профессии** — интерактивный опрос (интересы + навыки) с персональными рекомендациями (3 профессии).
* **Каталог профессий** — просмотр полного списка и детальной информации по каждой профессии с возможностью добавления в избранное.
* **Избранное** — возможность сохранять понравившиеся профессии и просматривать их позже.
* **Статистика** — наглядная статистика по категориям профессий: средняя зарплата и количество профессий в каждой категории, а также общие показатели.
* **Поиск** — поиск профессий по ключевым словам в названии или описании (вывод до 5 результатов).
* **Обратная связь** — отправка отзывов и предложений разработчикам с фиксацией времени, ID пользователя и текста сообщения.
* **Помощь** — справка по использованию бота с описанием всех функций.

## Установка и запуск

### Предварительные требования

* Python 3.8 или выше.
* Установленные библиотеки: `pyTelegramBotAPI`, `pandas`, `json`.
* Доступ к интернету для подключения к Telegram API.
* Базовые навыки работы с командной строкой.

### Установка зависимостей

1. Клонируйте репозиторий:
   ```bash
   git clone <https://github.com/Kirillitha/M8L1-M8L4.git>

Установите зависимости:

Следуйте инструкциям: задайте имя и username бота.

Затем в bot.py импортируйте его:

Запуск
Запустите скрипт:

bash

В консоли должно появиться сообщение:

Теперь найдите вашего бота в Telegram по username и отправьте команду /start.

Структура CSV‑файла (professions.csv)
Файл должен содержать следующие колонки:

Колонка	Тип данных	Описание	Пример
id	Целое число	Уникальный идентификатор профессии	1
profession	Строка	Название профессии	Веб‑разработчик
description	Строка	Краткое описание профессии (1–2 предложения)	Создаёт и поддерживает веб‑сайты и приложения
salary	Целое число	Средняя зарплата в рублях	120000
education	Строка	Требуемый уровень образования	Высшее техническое
skills	Строка	Ключевые навыки для профессии (через запятую)	Программирование, работа в команде
demand	Строка	Уровень востребованности	Высокая
prospects	Строка	Карьерные перспективы	Высокая
category	Строка	Категория профессии. Должна соответствовать ключам в category_mapping в коде	IT

1,Веб‑разработчик,Создаёт и поддерживает веб‑сайты и приложения,120000,Высшее техническое,Программирование, высокая,Высокая,IT
2,Графический дизайнер,Создаёт визуальные концепции для брендов,80000,Среднее специальное или курсы,Креативность,Дизайн,Средняя,Высокая,Творчество
Структура проекта
bot.py — основной код бота, содержит всю логику обработки команд и сообщений.

professions.csv — база данных профессий. Главный источник данных для рекомендаций и поиска.

favorites.json — хранилище избранных профессий пользователей. Хранит пары user_id: [prof_id1, prof_id2, ...].

feedback.txt — файл для записи отзывов пользователей. Каждая строка: дата, ID пользователя, имя, текст отзыва.

config.py (опционально) — файл конфигурации, куда вынесен токен бота и другие настройки.

Использование
После запуска и отправки команды /start бот покажет главное меню с кнопками:

🔎 Найти профессию — начать опрос для получения персональных рекомендаций. Бот задаст 2 вопроса:

Что вам больше нравится? (варианты: «💻 Техника», «🎨 Творчество», «👥 Работа с людьми», «📊 Аналитика»).

Какие у вас сильные стороны? (варианты: «💡 Креативность», «🧠 Логика», «💬 Коммуникация», «🔢 Аналитика»).
На основе ответов бот подберёт 3 подходящие профессии.

📖 Узнать о профессиях подробнее — просмотреть полный список профессий в виде inline‑кнопок. При нажатии на кнопку с профессией откроется детальная информация и кнопка «⭐ Добавить в избранное».

⭐ Избранное — посмотреть сохранённые профессии. Если список пуст, бот сообщит об этом.

📊 Статистика — ознакомиться со статистикой по категориям профессий (средняя зарплата, количество) и общими показателями (всего профессий, средняя зарплата по всем).

ℹ️ Помощь — открыть справку с описанием всех функций бота.

💬 Обратная связь — отправить отзыв разработчикам. Бот запросит текст сообщения, после отправки вернёт в главное меню.

Также можно искать профессии, просто написав ключевое слово (например, «программист» или «дизайн»). Бот покажет до 5 подходящих результатов.

Известные проблемы и решения
«Критическая ошибка соединения»: проверьте интернет‑соединение и правильность токена бота. Убедитесь, что прокси не блокирует запросы к Telegram API.

Пустой список профессий: проверьте, что файл professions.csv существует и имеет корректную структуру.

Ошибки при добавлении в избранное: проверьте права на запись в директории проекта для файлов favorites.json.

Бот не отвечает: убедитесь, что скрипт bot.py запущен и не завершился с ошибкой. Проверьте логи консоли.
